# گزارش پروژه: خواندن ورودی کیپد ماتریسی 4×4 با آردوینو (4×4 Keypad Input Reading)

## ۱. مقدمه  
در این پروژه， یک **کیپد ماتریسی 4×4** به آردوینو متصل شده است و با فشار دادن هر دکمه، کد آن (مثلاً `'1'`، `'A'`، `'#'`) در Serial Monitor نمایش داده می‌شود. 

## ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
- کیپد ماتریسی 4×4 (با دکمه‌های عددی و حرفی مثل `1–9`, `0`, `*`, `#`, `A–D`, `F1–F4`)  
- برد برد (Breadboard)  
- سیم‌های جامپر (jumper wires)

>.

## ۴. نحوه‌ی ساخت مدار  
- **کیپد 4×4**:  
  - ۴ پین سطر (Rows) → به پین‌های دیجیتال **2, 3, 4, 5** آردوینو  
  - ۴ پین ستون (Columns) → به پین‌های دیجیتال **6, 7, 8, 9** آردوینو  


```cpp
byte rowPins[ROWS] = {2, 3, 4, 5};   // سطرها
byte colPins[COLS] = {6, 7, 8, 9};   // ستون‌ها
```

## ۵. کد برنامه (اسکچ)  
کد مورد استفاده در این پروژه به شرح زیر است:

```cpp
#include <Keypad.h>   // اضافه کردن کتابخانه Keypad

const byte ROWS = 4;
const byte COLS = 4;

// تعریف الگوی کلیدها (4 سطر × 4 ستون)
char keys[ROWS][COLS] = {
  {'1', '2', '3', 'A'},
  {'4', '5', '6', 'B'},
  {'7', '8', '9', 'C'},
  {'*', '0', '#', 'D'}
};

// تعریف پین‌های سطر و ستون
byte rowPins[ROWS] = {2, 3, 4, };   // سطرها: پین‌های 2, 3, 4, 5
byte colPins[COLS] = {6, 7, 8, 9};   // ستون‌ها: پین‌های 6, 7, 8, 9

// ایجاد شیء Keypad
Keypad kpd = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS);

void setup() {
  Serial.begin(9600);   // شروع ارتباط سریال برای نمایش داده
}

void loop() {
  char key = kpd.getKey();   // خواندن کلید فشرده‌شده
  if (key) {                // اگر کلیدی فشرده شده باشد
    Serial.print("Pressed: ");
    Serial.println(key);     // نمایش کاراکتر در Serial Monitor
  }
}
```


## ۶. نحوه‌ی کارکرد  
- پس از آپلود کد, با فشار دادن هر دکمه روی کیپد، کاراکتر مربوطه در Serial Monitor نمایش داده می‌شود.  
- مثلاً فشار دادن دکمه `5` → خروجی: `Pressed: 5`  
- فشار دادن دکمه `C` → خروجی: `Pressed: C`  
- فشار دادن دکمه `#` → خروجی: `Pressed: #`  

## ۷. نتایج  
کیپد به درستی کلیدهای فشرده‌شده را تشخیص می‌دهد و در Serial Monitor نمایش می‌دهد.
